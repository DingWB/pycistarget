Installation
------------

To install pycistarget

.. code-block:: console

   (.venv) $ git clone https://github.com/aertslab/pycistarget.git
   (.venv) $ cd pycistarget
   (.venv) $ pip install -e .

Check your pycistarget version by running:

>>> import pycistarget
>>> pycistarget.__version__
